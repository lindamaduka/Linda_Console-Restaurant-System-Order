# Linda_Console Restaurant System Order

Restaurant Order System
A Dart console application that simulates a restaurant ordering system. A waiter can take orders from customers, add or remove items, and print a final receipt.

What It Does

Displays a menu of food and drink items
Lets you add multiple items to a customer's order
Lets you remove an item from the order by name
Prints a full receipt showing every item and the total amount due
Resets the order after billing so the next customer can start fresh
Runs in a loop until you choose to exit
